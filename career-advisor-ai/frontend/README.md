# Frontend (React + TypeScript + Vite)

Career Advisor dashboard client for:

- profile input
- career analysis output
- job matches, skill gaps, roadmap, resume and safety sections
- chat with career coach
- optional deep-research result panel
- LangGraph execution graph + trace view

## Run Locally

1. `npm install`
2. `npm run dev`

## Build

- `npm run build`

The frontend expects backend API at `http://localhost:8000` by default (see `src/api.ts`).
